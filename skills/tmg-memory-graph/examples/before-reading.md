# 示例：阅读前预存（即将开始阅读）

当你获取到 PDF / 网页 / 文件准备阅读时，**先把完整内容存入记忆**，再开始分析。这样记忆与真实时间线同步，忠实记录「此时开始读这份文档」。

## 请求体（POST /api/remember）

```json
{
  "text": "[2026-03-10 09:15] Kael — 准备阅读\n\n用户分享了一篇 PDF《Attention Is All You Need》，我下载并提取了全部文字。现在存入记忆后开始阅读和分析。\n\n以下是文档完整内容：\n\n---\n{PDF 提取的全部文字}\n---",
  "source_name": "论文-Attention",
  "event_time": "2026-03-10T09:15:00"
}
```

要点：时间用「开始阅读」的时刻；前缀说明文档来源与动作；`---` 内是完整原文。读完后可再发一条带总结的 remember（参考 read-document.md）。
