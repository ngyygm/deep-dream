# 记忆文本格式模板

每次 remember 的 `text` 应是一段**按时间排列的忠实叙事**，并带上下文前缀。

## 格式

```
[{时间}] {Agent名称} — {活动类型}

{内容}
```

- **时间**：事件发生的时间或时间段，如 `2026-03-10 14:00` 或 `2026-03-10 14:00-16:00`
- **Agent名称**：你的名字/身份标识
- **活动类型**：如「工作记录」「阅读论文」「当日反思」「准备阅读」
- **内容**：按时间点或逻辑组织的叙事；若是文档，用 `---` 包裹完整原文

## 与 event_time 的关系

请求体中的 `event_time`（ISO 8601）应与文本前缀中的「时间」一致，用于图谱中实体/关系的 `physical_time`。
